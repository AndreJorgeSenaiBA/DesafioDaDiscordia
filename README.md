# DesafioDaDiscordia
Desafio Da Discordia é um desafio para programar um sistema aleatório em tecnologia de minha preferencia sem IA, contra meus amigos de curso!

IDEIA:
tem que criar um jogo onde precisa de 2 jogadores ou mais, e cada jogador deve girar o dado, e aleatoriamente cair num valor do dado, o jogador com a maior pontuação vence a rodada e ganha 1 ponto, no final de 5 rodadas, o jogador com mais pontos vence!
em caso de empate nas rodadas, os jogadores empatados devem jogar o dado para desempate!
